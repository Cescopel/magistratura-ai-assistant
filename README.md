\# Assistente Magistratura AI



Assistente AI per la preparazione al concorso di magistratura.



\## Features

\- Upload documenti Word/PDF

\- Q\&A intelligente con AI

\- Ricerca automatica nei documenti

\- Chunking per file grandi



\## Setup Locale

1\. `pip install -r requirements.txt`

2\. Crea file `.env` con `ANTHROPIC\_API\_KEY=your-key`

3\. `streamlit run app.py`
